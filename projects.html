Software Engineering

Projects by Students

Senior Projects

Perfusion Device

The Perfusion Device was a collaboration project between senior software and mechanical engineers. The device is used to deliver blood to a capillary bed in its biological tissue using sensors and monitors. This project counted towards SER491 and SER492.

Junior Projects

QUBER

QUBER was a prototype designed by two juniors based on requirements declared by a customer. QUBER serves as a median to connect students around Quinnipiac into finding rides. The prototype counted towards SER340: Software Requirements Analysis.

Sophomore Projects

Archer Faceoff

Archer Faceoff was a Java based game developed in the early 2000s at Quinnipiac. The game is a turn-based game featuring two players shooting arrows at eachother until one of them dies. Students enrolled in SER225 were required to use a scrum cycle to enhance the game and fix any existing bugs.

Freshman Projects

Scala Game

Freshman enrolled in CSC110 are required to use Scala to create a computer game. Students are encouraged to be as creative as possible and make the game fun and entertaining for users.
